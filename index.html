Daftar File Data dan Visualisasi

Halaman ini berisi kumpulan file data dan visualisasi yang digunakan dalam proyek Strategi Algoritma – Tugas Besar. Setiap file dapat diakses melalui hyperlink pada kolom “Tautan”.

No Nama File Jenis Keterangan Tautan
1 1_peta_lokasi_titik.html HTML (Visualisasi) Peta interaktif yang menampilkan lokasi titik-titik penting (misalnya titik evakuasi, lokasi rawan, atau titik referensi). Buka Halaman
2 2_peta_visualisasi_jarak_konektivitas.html HTML (Visualisasi) Visualisasi peta yang menunjukkan jarak dan konektivitas antar titik berdasarkan matriks jarak jalan. Buka Halaman
3 matriks_jarak_jalan_asli.csv CSV (Data) File matriks jarak antar titik berdasarkan jarak jalan sebenarnya, digunakan sebagai input perhitungan algoritma. Unduh File
4 flood_data.xls Excel (Data) Data terkait banjir (misalnya area terdampak, level risiko, atau informasi historis kejadian banjir). Unduh File
5 evac_data.xls Excel (Data) Data lokasi dan informasi titik evakuasi yang digunakan dalam analisis dan visualisasi. Unduh File
6 data_gabungan_cilacap.csv CSV (Data) Dataset gabungan wilayah Cilacap yang mengintegrasikan data banjir, evakuasi, dan informasi pendukung lainnya. Unduh File
Catatan:
Pastikan seluruh file berada pada root repository GitHub Pages (StrategiAlgoritma-TugasBesar) agar hyperlink dapat diakses dengan benar.
